# よく使う操作

## Create new post

```bash
./scripts/new_post.sh
```

もしくは
```bash
hugo new posts/POST-NAME.md
```

## Run server

```bash
hugo server -D
```

## Build

```bash
hugo
```

## Deploy

GitHubのmainブランチにpushすると自動でデプロイされる。
